deeprob.torch.routines module¶

deeprob.torch.routines.train_model(model, data_train, data_val, setting, lr=0.001, batch_size=100, epochs=1000, patience=30, optimizer='adam', optimizer_kwargs=None, weight_decay=0.0, train_base=True, class_weights=None, num_workers=0, device=None, verbose=True)¶

Train a Torch model.

Parameters

  • model – The model to train.

  • data_train – The train dataset.

  • data_val – The validation dataset.

  • setting – The train setting. It can be either ‘generative’ or ‘discriminative’.

  • lr – The learning rate to use.

  • batch_size – The batch size for both train and validation.

  • epochs – The number of epochs.

  • patience – The epochs patience for early stopping.

  • optimizer – The optimizer to use.

  • optimizer_kwargs – A dictionary containing additional optimizer parameters.

  • weight_decay – L2 regularization factor.

  • train_base – Whether to train the input base module. Only applicable for normalizing flows.

  • class_weights – The class weights (for im-balanced datasets). Used only if setting=’discriminative’.

  • num_workers – The number of workers for data loading.

  • device – The device used for training. If it’s None ‘cuda’ will be used, if available.

  • verbose – Whether to enable verbose mode.

Returns

The train history.

deeprob.torch.routines.test_model(model, data_test, setting, batch_size=100, num_workers=0, class_weights=None, device=None)¶

Test a Torch model.

Parameters

  • model – The model to test.

  • data_test – The test dataset.

  • setting – The test setting. It can be either ‘generative’ or ‘discriminative’.

  • batch_size – The batch size for testing.

  • num_workers – The number of workers for data loading.

  • class_weights – The class weights (for im-balanced datasets). Used only if setting=’discriminative’.

  • device – The device used for training. If it’s None ‘cuda’ will be used, if available.

Returns

The mean log-likelihood and two standard deviations if setting=’generative’. The negative log-likelihood and classification metrics if setting=’discriminative’.
