spnflow.learning.structure.learn_structure(data, distributions, domains, learn_leaf='mle', learn_leaf_params=None, split_rows='kmeans', split_cols='rdc', split_rows_kwargs=None, split_cols_kwargs=None, min_rows_slice=256, min_cols_slice=2, verbose=True)¶

Learn the structure and parameters of a SPN given some training data and several hyperparameters.

Parameters

  • data – The training data.

  • distributions – A list of distributions classes (one for each feature).

  • domains – A list of domains (one for each feature).

  • learn_leaf – The method to use to learn a distribution leaf node (it can be ‘mle’ or ‘isotonic’).

  • learn_leaf_params – The parameters of the learn leaf method.

  • split_rows – The rows splitting method (it can be ‘kmeans’, ‘gmm’, ‘rdc’ or ‘random’).

  • split_cols – The columns splitting method (it can be ‘gvs’, ‘rdc’ or ‘random’).

  • split_rows_kwargs – The parameters of the rows splitting method.

  • split_cols_kwargs – The parameters of the cols splitting method.

  • min_rows_slice – The minimum number of samples required to split horizontally.

  • min_cols_slice – The minimum number of features required to split vertically.

  • verbose – Whether to enable verbose mode.

Returns

A learned valid SPN.
