spnflow.algorithms.evaluation.eval_bottom_up(root, x, leaf_func, node_func, return_results=False)¶

Evaluate the SPN bottom up given some inputs and leaves and nodes evaluation functions.

Parameters

  • root – The root of the SPN.

  • x – The inputs.

  • leaf_func – The function to compute at the leaves.

  • node_func – The function to compute at each internal node.

  • return_results – A flag indicating if this function must return the log likelihoods of each node of the SPN.

Returns

The outputs. Additionally it returns the output of each node.

spnflow.algorithms.evaluation.eval_top_down(root, x, ls, leaf_func, sum_func)¶

Evaluate the SPN top down given some inputs, the likelihoods of each node and a leaves evaluation function. The leaves to evaluate are chosen by following the nodes having maximum likelihood top down.

Parameters

  • root – The root of the SPN.

  • x – The inputs (must have at least one NaN value).

  • ls – The likelihoods of each node.

  • leaf_func – The leaves evaluation function.

  • sum_func – The sum node evaluation function.

Returns

The NaN-filled inputs.
