SPNFlow¶

Introduction¶

SPNFlow is a python implementation of Sum-Product Networks (SPNs) learning and inference. It also contains an implementation of Randomized and Tensorized SPNs (RAT-SPNs) using Tensorflow 2.0. In this repository it is also present a novel model that combines RAT-SPNs and Masked Autoregressive Flows (MAFs) for density estimation.

Features¶

  • Inference algorithms for SPNs. [1] [2] [4]

  • Learning algorithms for SPNs structure. [1] [2] [3] [4]

  • Optimization of the structure of SPNs. [4]

  • JSON I/O operations for SPNs. [4]

  • Implementation of RAT-SPN using Keras and Tensorflow 2.0. [5]

  • Implementation of RAT-SPN combined with MAFs using Keras and Tensorflow 2.0. [5] [6] [7]
